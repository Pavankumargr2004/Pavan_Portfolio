## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/portfolio.git
```

2. Navigate to the project folder:

```bash
cd portfolio
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm start
```

5. Open your browser and visit:

```bash
http://localhost:3000
```
